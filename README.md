# YouTube Content Creation Analytics Dashboard

This project is an interactive dashboard built with **Python, Streamlit, and Plotly** to analyze YouTube content performance.

## Features

* Upload and analyze YouTube content datasets
* Interactive visualizations (area charts, pie charts, engagement maps)
* Question-based analysis input
* Modern and colorful dashboard UI

## Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## How to Run

1. Install requirements
   pip install -r requirements.txt

2. Run the app
   streamlit run app.py

## Project Purpose

This dashboard helps analyze engagement metrics like views, likes, comments, and shares for YouTube content creators.

Built for a **Hackathon Project**.
